 🎯 Number Guessing Game (Java)

A simple console-based **Number Guessing Game** built using Java. The program generates a random number, and the player has to guess it using the hints provided until the correct number is guessed.

 📌 Features

- 🎲 Random number generation
- 🔢 Accepts user input using the Scanner class
- 📉 Displays "Too Low" if the guess is smaller than the generated number
- 📈 Displays "Too High" if the guess is greater than the generated number
- ✅ Congratulates the player when the correct number is guessed
- 🔁 Continues until the player guesses the correct number
- 💻 Beginner-friendly Java project
  
 🛠️ Technologies Used

- Random Class

 🎮 Gameplay

1. The program generates a random number.
2. The player enters a guess.
3. The game provides feedback:
   - 📉 Too Low
   - 📈 Too High
4. The player continues guessing until the correct number is found.
5. The game displays the number of attempts taken to guess the correct number.
